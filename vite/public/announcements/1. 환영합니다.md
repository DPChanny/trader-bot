# 환영합니다

안녕하세요. Trader Bot 운영팀입니다.

운영팀에서 신규 사용자분들께 환영 인사를 드립니다.

Trader Bot은 Discord 서버 내전 운영을 간단하게 시작하고 반복할 수 있도록 운영팀이 준비한 서비스입니다.
공식 Trader Bot 서버에 참가하면 운영팀 공지와 업데이트 소식을 더 빠르게 확인할 수 있습니다.

## 함께해요

- Discord 서버에 Trader Bot을 초대하고 바로 시작해보시길 운영팀이 권장드립니다.
- 서버 멤버와 프리셋을 구성해 경매를 간단하게 운영해보세요.
- 사용 중 불편한 점이나 원하는 기능은 언제든 운영팀에 의견으로 남겨주세요.
- 공식 Trader Bot 서버에서 운영팀 공지, 업데이트, 운영 팁을 함께 확인해주세요.

## 안내

- 운영팀은 많은 참여를 부탁드립니다.
- 여러분의 참여가 Trader Bot을 더 좋은 서비스로 만드는 데 큰 도움이 됩니다.

## 주의 사항

- Trader Bot이 서버에 초대되어야 해당 서버가 서비스에 등록됩니다.
- 소유자 역할은 Trader Bot이 자동으로 관리합니다.
- 서버 관리자 권한이 있는 멤버에게는 관리자 역할이 자동 부여됩니다.
- 서버 관리자 권한을 잃으면 편집자를 제외한 멤버에게 열람자 역할이 자동 부여됩니다.
- 일반 멤버에게는 열람자 역할이 자동 부여됩니다.
- 소유자 역할을 제외한 역할은 관리자 역할부터 부여할 수 있습니다.
- 편집자 역할부터 서버 멤버 및 프리셋 멤버 정보를 수정할 수 있습니다.
- 관리자 역할부터 프리셋을 구성할 수 있습니다.
