# 패치 계획 - v0.4.1 - 확장

## 개선사항

- 리스트 가독성과 조작 효율을 높이기 위해 그리드 정렬 방식을 개선할 계획입니다.

## 추가사항

- 운영 서버 연계를 위해 webhook 연동 기능을 추가할 계획입니다.
- 멤버 정보 URL을 빠르게 확인할 수 있도록 iframe 미리보기 기능을 추가할 계획입니다.
- 경매 결과를 확인하고 활용할 수 있도록 결과 조회 기능을 추가할 계획입니다.
- 운영 편의성을 높이기 위해 봇 명령어를 확장할 계획입니다.

## 확장사항

- v0.4.0 필수 편의 기능을 기반으로 운영 확장 기능을 단계적으로 적용할 계획입니다.
