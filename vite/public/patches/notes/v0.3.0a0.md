# 패치 노트 - v0.3.0a0 - 안정화

## 개선사항

- 프론트엔드 구조를 Vite 기반으로 재정비해 개발 및 빌드 효율을 개선했습니다.
- 로그 스키마와 로그 레벨 체계를 정리해 운영 중 원인 추적 흐름을 개선했습니다.
- 스크롤, 레이아웃, 스타일 토큰을 조정해 화면 사용성과 시각 일관성을 개선했습니다.

## 추가사항

- 패치 페이지 및 문서 공개 흐름을 추가해 버전별 계획과 노트를 조회할 수 있도록 구성했습니다.
- 경매, 포지션, 프리셋 관련 라우터와 서비스 구성을 확장해 기능 기반을 추가했습니다.
- 배포 자동화를 위해 프론트엔드, 백엔드, 인프라 워크플로우를 추가했습니다.

## 안정화사항

- Base64 처리 및 로그 마스킹 관련 오류를 수정해 런타임 안정성을 개선했습니다.
- 개발 및 배포 과정에서 발생한 다수의 버그를 수정해 기능 동작 신뢰성을 높였습니다.

## 인프라사항

- Nginx, PM2, CloudWatch 에이전트 설정을 정비해 운영 환경 구성을 개선했습니다.
- CI/CD 의존성과 실행 순서를 정리해 배포 파이프라인 안정성을 개선했습니다.

## 문서사항

- README 및 환경 변수 가이드를 보강해 온보딩과 운영 문서 가독성을 개선했습니다.
- LICENSE를 추가해 프로젝트 라이선스 정보를 명확히 했습니다.
