# 환영합니다!

안녕하세요! Trader Bot 개발팀입니다.
Trader Bot을 새롭게 찾아주신 여러분을 진심으로 환영합니다.

Trader Bot은 Discord 서버 내전 및 참가자 경매를 쉽고 빠르고 공정하게 진행할 수 있도록 돕는 서비스입니다.

## 공식 Discord 서버에서 소통해요

새로운 배포 소식, 기능 건의사항, 혹은 사용 중 발생한 버그 등은 언제든지 **공식 Discord 서버**에 방문하셔서 개발자들과 직접 소통해 주세요!
여러분의 생생한 피드백과 다양한 아이디어가 Trader Bot을 더욱 멋진 서비스로 만들어갑니다.

## 공식 프리셋을 적극 활용해 보세요

어떤 방식으로 팀원들의 티어와 포지션 설정을 구성해야 할지 막막하신가요?
시스템에서 기본적으로 제공하는 **공식 프리셋**을 먼저 사용해 보는 것을 적극 권장합니다!
복잡한 초기 세팅 및 타이핑 없이도, 우리 서버에 꼭 맞는 경매와 내전을 즉시 완성하고 시작하실 수 있습니다.

## 주의사항

- Trader Bot 봇이 여러분의 디스코드 서버에 먼저 초대되어 있어야만 해당 서버를 본 웹 서비스에서 연동해 이용할 수 있습니다!

앞으로도 더 눈높이에 맞고 편리한 서비스를 제공해 드리기 위해 쉴 틈 없이 고민하고 발전하겠습니다.
