# Trader Bot의 동작과 역할의 권한

안녕하세요. Trader Bot 운영팀입니다.

먼저 Trader Bot을 서버에 초대해 주시고 이용해 주셔서 진심으로 감사드립니다!

처음 서비스를 이용하시면서 "누가 어떤 권한을 가지게 되는지", 혹은 "왜 내 역할이 자동으로 바뀌었는지" 헷갈리시는 분들이 많으실 텐데요. 원활하고 안전한 시스템 이용을 위해 Trader Bot의 자동 역할 관리 시스템과 권한별 세부 기능을 안내해 드립니다.

특히 디스코드 서버 자체의 권한과 Trader Bot 내부의 역할이 어떻게 연동되는지 꼭 확인해 주세요!

## 1. Trader Bot의 동작

- 소유자 역할은 Trader Bot이 자동으로 관리합니다.
- 서버 관리자 권한이 있는 멤버에게는 관리자 역할이 자동 부여됩니다.
- 서버 관리자 권한을 잃으면 편집자를 제외한 멤버에게 열람자 역할이 자동 부여됩니다.
- 일반 멤버에게는 열람자 역할이 자동 부여됩니다.

## 2. 역할의 권한

- 소유자 역할을 제외한 역할은 관리자 역할부터 부여할 수 있습니다.
- 편집자 역할부터 서버 멤버 및 프리셋 멤버 정보를 수정할 수 있습니다.
- 관리자 역할부터 프리셋을 구성할 수 있습니다.
