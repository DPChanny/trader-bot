# v0.7.0

안녕하십니까. Trader Bot 운영팀입니다.

v0.7.0 패치 계획은 **PRO 전용 기능 확장 및 운영 편의 고도화**에 목적이 있습니다.

## 추가사항

- **정보 링크:** 외부 웹사이트(OP.GG, PS 등)의 정보를 경매 화면에서 직접 조회할 수 있는 정보 링크 기능을 추가합니다.
- **경매 결과 다운로드:** 경매 종료 후 결과 데이터를 파일로 내보낼 수 있습니다.

## 참고사항

- 다양한 봇 명령어를 기획 중입니다. 원하는 기능이나 아이디어가 있다면 공식 디스코드 서버에 의견을 남겨주세요.
