# v0.5.0

안녕하십니까. Trader Bot 운영팀입니다. v0.5.0 버전의 패치 계획은 다양한 기기 환경에서 일관된 사용 경험을 제공하기 위한 UI 구조 정비를 목표로 합니다.

## 추가사항

- 모바일 환경 대응을 위해 반응형 레이아웃을 본격 적용할 계획입니다.

## 개선사항

- 범용 사용성을 높이기 위해 전체 레이아웃 구조를 정리할 계획입니다.
- 다양한 화면 크기와 사용 시나리오에서 일관된 사용 경험을 제공할 수 있도록 UI 기준을 정비할 계획입니다.
- 유지보수성과 일관성을 높이기 위해 CSS 토큰 체계를 개선할 계획입니다.

## 조정사항

- 화면별 반응형 기준을 운영 가이드와 함께 정리해 이후 기능 추가 시에도 일관성을 유지할 계획입니다.
