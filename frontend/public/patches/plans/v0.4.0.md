# v0.4.0

안녕하십니까. Trader Bot 운영팀입니다.

v0.4.0 패치 계획은 **목록 탐색 효율 활성화 및 프리셋 작성 워크플로 고도화**를 통해 서비스 관리 시 소요되는 반복적인 운영 작업을 최소화하는 것에 목적이 있습니다.

## 추가사항

- **프리셋 작성 파이프라인 단축:** 기존 프리셋 복제 기능 및 기본 템플릿에서의 파생 생성 기능 도입
- **운영 템플릿 제공:** 시스템에서 공식적으로 제공하는 표준 운영 프리셋 신설
- **탐색 편의 기능 탑재:** 목록 페이지네이션, 고급 정렬, 그리고 통합 검색 기능 도입

## 참고사항

- 프리셋 복제 프로세스의 도입으로 관리자는 신규 프리셋 생성 시 초기 세팅에 소요되는 시간을 대폭 절감할 수 있게 됩니다.
- 검색 및 정렬 기능은 대량의 데이터 처리 시에도 지연 없이 동작할 수 있도록 서버 사이드 페이지네이션(Cursor 또는 Offset 방식)과 연동될 예정입니다.
