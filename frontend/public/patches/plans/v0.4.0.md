# v0.4.0

안녕하십니까. Trader Bot 운영팀입니다. v0.4.0 버전의 패치 계획은 목록 탐색과 프리셋 생성 과정의 반복 업무를 줄이기 위한 개선을 목표로 합니다.

## 추가사항

- 기존 프리셋 복사 및 기반 프리셋 생성 흐름을 지원해 프리셋 생성 과정을 단순화할 계획입니다.
- 프리셋 템플릿 기능을 도입해 운영 서버의 공식 프리셋 흐름을 참고할 수 있도록 구성할 계획입니다.

## 개선사항

- 목록 조회 편의성을 높이기 위해 페이지네이션, 정렬, 검색 기능을 통합할 계획입니다.
- 필수 편의 기능을 우선 적용해 실사용 시나리오에서 반복 업무 부담을 줄일 계획입니다.

## 조정사항

- 운영 흐름에서 자주 사용하는 조회 및 생성 동선을 기준으로 적용 우선순위를 조정할 계획입니다.
