# v0.4.1

안녕하십니까. Trader Bot 운영팀입니다. v0.4.1 버전의 패치 계획은 운영 연계 기능을 확장하고 리스트 조작 효율을 높여 실무 대응 속도를 개선하는 것을 목표로 합니다.

## 추가사항

- 경매 결과를 확인하고 활용할 수 있도록 결과 조회 기능을 추가할 계획입니다.
- 멤버 정보 링크를 빠르게 확인할 수 있도록 iframe 미리보기 기능을 도입할 계획입니다.
- 운영 편의성을 높이기 위해 봇 명령어를 확장할 계획입니다.
- 운영 서버 연계를 위해 웹훅 연동 기능을 추가할 계획입니다.

## 개선사항

- 리스트 가독성과 조작 효율을 높이기 위해 그리드 정렬 방식을 개선할 계획입니다.

## 조정사항

- 운영팀의 사용 빈도가 높은 연계 기능부터 순차 배포해 적용 리스크를 관리할 계획입니다.
- v0.4.0의 필수 편의 기능을 기반으로 운영 확장 기능을 단계적으로 반영할 계획입니다.
