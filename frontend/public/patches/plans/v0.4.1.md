# v0.4.1

안녕하십니까. Trader Bot 운영팀입니다. v0.4.1 버전의 패치 계획은 운영 연계 기능을 확장하고 목록 조작 효율을 개선해 실무 대응 속도를 높이는 데 목적이 있습니다.

## 추가사항

- 경매 종료 이후 결과 데이터를 즉시 검토하고 후속 작업에 활용할 수 있도록 결과 조회 기능을 추가할 계획입니다.
- 멤버 정보 링크를 별도 이동 없이 확인할 수 있도록 iframe 기반 미리보기 기능을 도입할 계획입니다.
- 운영 자동화 범위를 넓히기 위해 봇 명령어 기능을 도입할 계획입니다.
- 공식 서버와의 이벤트 연계를 지원하기 위해 웹훅 기능을 추가할 계획입니다.

## 개선사항

- 리스트의 가독성과 조작 일관성을 높이기 위해 그리드 정렬 인터랙션을 개선할 계획입니다.
