# v0.6.0

안녕하십니까. Trader Bot 운영팀입니다.

v0.6.0 패치 계획은 **프론트엔드 리팩토링 및 전면 재디자인, 모바일 지원, 순서 지정 기능 추가**에 목적이 있습니다. 유저 피드백을 반영하는 과정에서 기존 프론트엔드 코드의 기술 부채가 한계에 달해, 새로운 기능을 새 UI에 맞게 함께 구현합니다.

## 추가사항

- **모바일 지원:** 모바일 환경에서도 서비스를 원활하게 이용할 수 있도록 반응형 레이아웃을 지원합니다.
- **순서 지정:** 경매 순서, 포지션 순서 등 다양한 항목의 순서를 직접 지정할 수 있습니다.

## 개선사항

- **프론트엔드 리팩토링:** 기존 스타일링 코드를 Tailwind CSS + shadcn/ui 기반으로 전면 재작성합니다. 레거시 코드로 인한 기술 부채를 해소하고 향후 기능 확장의 기반을 마련합니다.
- **페이지 재디자인:** 전체 페이지의 UI를 새 디자인 시스템에 맞게 재설계합니다.
- **UX 개편:** 유저 피드백을 바탕으로 주요 흐름의 UX를 개선합니다.
