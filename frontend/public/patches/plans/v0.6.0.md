# v0.6.0

안녕하십니까. Trader Bot 운영팀입니다.

v0.7.0 패치 계획은 **운영 연계 기능 집중 확장 및 관리 편의 고도화**를 핵심으로, 경매 사후 데이터 접근성을 높이고 외부 채널 연동 파이프라인을 신설하여 실무 관리 대응 속도를 높이는 데 목적이 있습니다.

## 추가사항

- **사후 리뷰 편의 강화:** 경매 종료 후 즉시 결과 데이터를 리뷰하고 모니터링할 수 있는 독립적 결과 조회 기능 신설
- **외부 채널 연동 파이프라인 신설:** 디스코드 공식 채널과 메인 이벤트 연계를 위한 웹훅(Webhook) 발송 지원
- **신속한 멤버십 확인:** 외부 사이트 이동 없이 인앱 환경에서 멤버 정보를 미리보는 Iframe 기반 뷰어 도입
- **운영 템플릿 제공:** 시스템에서 공식적으로 제공하는 표준 운영 프리셋 신설 및 기본 템플릿 기반 파생 생성 기능 도입

## 개선사항

- **운영 자동화 연동 확대:** 디스코드 슬래시 명령어(/) 기반 봇 운영 제어 환경 추가 확대

## 참고사항

- 추가되는 봇 명령어와 인앱 웹훅 기능은 강력한 제어 시스템과 연동되므로, 역할의 권한이 부여된 지정된 관리 인원만 사용하도록 제어됩니다.
- 운영 템플릿은 시스템 수준에서 사전 정의된 구조로 제공되며, 관리자가 이를 기반으로 파생 프리셋을 즉시 생성할 수 있도록 지원합니다.
