# v0.5.2

시스템 안정성 개선에 집중한 버전입니다. 사용자에게 직접 노출되는 변경은 없으나, 경매 서버의 재시작 반복 현상을 유발하던 근본 원인을 제거했습니다.

## 수정

- **경매 타임아웃 버그 수정:** 경매 시간이 만료될 때 Redis에 고아 키가 생성되어 서버가 반복 재시작되던 문제를 수정했습니다. 타임아웃 시 상태 저장 없이 만료 이벤트만 발행하도록 처리 방식을 변경했습니다.
- **Redis 상태 업데이트 원자성 보강:** 경매 키가 이미 만료된 상태에서 상태 값이 기록되지 않도록 Lua 스크립트 수준의 존재 여부 확인 절차를 추가했습니다.
- **배포 자동화 보완:** 결제 관련 환경 변수가 배포 워크플로우에 누락되어 있던 항목을 추가했습니다.
