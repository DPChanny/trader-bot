# v0.3.0

안녕하십니까. Trader Bot 운영팀입니다. v0.3.0 버전은 운영 안정성 확보를 목표로 배포 흐름 표준화와 인프라 구조 개편을 중심으로 출시했습니다.

## 추가사항

- 개발 흐름을 dev, beta, prod 단계로 확정하고, 필요 시 beta.trader-bot.org에 베타 버전을 선택적으로 출시할 수 있도록 운영 기준을 마련했습니다.
- IaC workflow를 도입해 인프라 변경 이력을 코드 기반으로 관리하고 장애 대응을 위한 기반을 확보했습니다.
- 패치 노트 체계를 추가해 릴리즈 변경 내역을 버전 단위로 일관되게 제공할 수 있도록 정비했습니다.
- 공지 체계를 추가해 운영 정책과 주요 변경 사항을 사용자에게 신속히 전달할 수 있도록 구성했습니다.

## 개선사항

- AWS 인프라를 전면 개편해 서비스 구성의 일관성과 확장성을 개선했습니다.
- GitHub Actions 기반 Workflow를 구축해 배포 자동화를 중심으로 배포 파이프라인의 일관성과 속도를 개선했습니다.
- 운영 관점의 릴리즈 기록 체계를 정비해 이슈 대응 및 변경 추적 효율을 높였습니다.

## 조정사항

- dev, beta, prod 환경별 반영 순서를 고정하고, beta.trader-bot.org에서의 선택적 사전 검증 후 상위 환경으로 승격하는 기준을 적용했습니다.
- IaC 기반 인프라 변경 절차를 사전 검토, 적용, 사후 확인 단계로 표준화해 배포 중단 가능성을 낮췄습니다.
- 패치 노트 및 공지를 릴리즈 시점에 동시 반영하는 원칙으로 조정해 사용자 커뮤니케이션 공백을 최소화했습니다.
