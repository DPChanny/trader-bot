# v0.5.3

결제 및 구독 관련 사용 경험을 개선하고, 결제 서비스 정책 요건을 충족하는 데 집중한 버전입니다.

## 추가

- **자동 결제 설정 섹션 신설:** 서버 페이지에서 플랜·결제 수단·자동 결제 해제를 한 화면에서 관리할 수 있게 됩니다. 설정 변경을 위해 별도 페이지로 이동할 필요가 없어졌습니다.
- **구독 업그레이드:** Plus 구독 중 Pro로 즉시 전환할 수 있습니다. 남은 기간을 반영한 차액만 청구됩니다.
- **다음 결제 플랜 예약:** 현재 구독 기간은 유지하면서 다음 주기부터 적용될 플랜을 미리 지정할 수 있습니다.

## 개선

- **결제창 종료 후 자동 처리:** 카드 등록 완료 후 별도 팝업이나 페이지 이동 없이 현재 화면에서 바로 반영됩니다.
- **플랜 표기 통일:** 플랜 선택 항목에 플랜명·월 금액·결제 주기가 함께 표시됩니다.

## 수정

- **결제창 취소 시 오류 모달 미노출:** Toss 결제창을 닫거나 취소했을 때 불필요한 오류 창이 뜨던 문제를 수정했습니다.
